# Cosmetic Scanner - Fixed

Displays overlay with info of how many cosmetic chests there are on the level and which rarity. Usefull for knowing if to reset a level to optimize coins farming. Should now work during multiplayer, at least for host player, needs testing.


## Features

- HUD overlay
- Live rescanning
- Toggle with F6
