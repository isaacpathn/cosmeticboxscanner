# Cosmetic Scanner - Fixed v1.0.4

Displays overlay with info of how many cosmetic chests there are on the level and which rarity. Usefull for knowing if to reset a level to optimize coins farming.

## Features

- HUD overlay
- Live rescanning
- Toggle with F6
